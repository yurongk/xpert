# Acknowledgments

Xpert AI would not be possible without the support and assistance of the open-source community and the companies who believe in giving back. This page recognizes the key technologies and services that power our platform.

---

## 🛠️ Core Technologies

### Containerization & Orchestration

**[Docker](https://www.docker.com)**

Docker is a software platform that allows developers to package and deploy applications inside containers. Containers are lightweight, portable, and self-sufficient environments that can run on any operating system or cloud platform.

**[Kubernetes](https://kubernetes.io)**

Kubernetes, also known as K8s, is an open-source system for automating deployment, scaling, and management of containerized applications.

---

### Frontend Framework

**[Angular](https://angular.dev)**

Angular is a platform and framework for building single-page client applications using HTML and TypeScript. Angular is written in TypeScript and implements core and optional functionality as a set of TypeScript libraries.

**[Tailwind CSS](https://tailwindcss.com)**

Tailwind CSS is a utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.

---

### Backend Framework

**[NestJS](https://nestjs.com)**

NestJS is a framework for building efficient, scalable Node.js server-side applications. It uses progressive JavaScript, is built with TypeScript, and combines elements of OOP, FP, and FRP.

**[TypeORM](https://typeorm.io)**

TypeORM is an ORM that can run in NodeJS, Browser, Cordova, PhoneGap, Ionic, React Native, NativeScript, Expo, and Electron platforms and can be used with TypeScript and JavaScript.

---

### AI & LLM

**[LangChain](https://js.langchain.com)**

LangChain is a framework for developing applications powered by language models. It enables applications that are context-aware and can reason.

---

### Data Visualization

**[Apache ECharts](https://echarts.apache.org)**

ECharts is a powerful, interactive charting and visualization library for browser, supporting various chart types with rich features.

---

### OLAP Engine

**[Mondrian](https://github.com/pentaho/mondrian)**

Mondrian is an OLAP (Online Analytical Processing) engine written in Java. It enables executing MDX queries and provides multi-dimensional analysis capabilities.
